# MULTIPLE_FORMULARIOS_9X9 🖥️

Este repositorio contiene la interfaz de usuario avanzada y el sistema de gestión multiventana para el análisis de Sudokus 9x9, desarrollado en Visual Basic 6.0.

## 🚀 Descripción

`MULTIPLE_FORMULARIOS_9X9` es una aplicación MDI (Multiple Document Interface) diseñada para ofrecer una experiencia de usuario robusta en el estudio de matrices 9x9. El sistema permite abrir y comparar simultáneamente múltiples soluciones, plantillas y procesos de cálculo, facilitando el análisis comparativo de datos matemáticos en tiempo real.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Arquitectura:** Interfaz de múltiples formularios (MDI) con gestión de estados.
- **Funcionalidades Clave:**
  - Visualización concurrente de múltiples grillas de 9x9.
  - Sincronización de datos entre formularios de análisis y resultados.
  - Paneles de herramientas integrados para la manipulación de plantillas.
  - Diseño modular que facilita la expansión de nuevas ventanas de diagnóstico.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Formularios (.frm), módulos (.bas) y el contenedor principal (.vbp).
- `/res`: Recursos visuales, iconos y archivos de configuración de la interfaz.
- `/docs`: Guía de usuario y descripción de los flujos de trabajo del sistema.
